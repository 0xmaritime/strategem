Choose Your Analysis Version

Strategem offers two distinct analysis approaches with different capabilities and requirements.

V1

Inferred Decision Focus

System infers decision context from materials. Ideal for exploratory analysis when the decision question is not clearly defined.

  • ✓ Optional decision question
  • ✓ Inferred decision focus
  • ✓ System-level claims allowed
  • ✓ Frameworks adapt to context
  • ✓ Simpler input requirements

V2 NEW

Decision-Bound & Option-Aware

Requires explicit decision context and options. Provides deeper analytical depth with cross-framework tension mapping.

  • ✓ Required decision question
  • ✓ Required options (min 2)
  • ✓ All claims option-aware
  • ✓ Cross-framework tension mapping
  • ✓ Sensitivity triggers
  • ✓ Structured JSON artefacts

Key Differences

Feature V1 (Inferred) V2 (Decision-Bound)
Decision Question Optional (inferred) Required
Options Optional Required (min 2)
Claim Scope System-level allowed Option-aware only
Tension Mapping Simple section Explicit with claim-level details
Sensitivity Triggers Basic surface First-class with inquiry prompts
Output Format Markdown report Structured JSON artefacts

System Boundaries (Both Versions)

This system does NOT:

  • Output decisions or recommendations
  • Rank or compare options
  • Optimize objectives
  • Provide investment advice
  • Validate against external reality
  • Learn from past outcomes
